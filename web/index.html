API Documentation

Telegram Scraper REST API — Endpoints & Examples

v1
🔐

Authentication

Every request must include your API key in the request headers.

Header x-api-key
Value The API_KEY value from your .env file
GET

/api/v1/messages

Fetch recent messages from any Telegram channel, group, or user you have access to.

Parameters

NameTypeDefaultDescription
target required string — Username, invite link, or ID (e.g. BinanceWaves)
limit integer 50 Max number of messages to return
search string — Filter messages by keyword
download_media boolean false If true, downloads photos/videos and returns URL

Example — cURL

bash 
curl -X GET "https://your-domain.com/api/v1/messages?target=BinanceWaves&limit=5" \
     -H "x-api-key: YOUR_API_KEY"

Example — Search with Media

bash 
curl -X GET "https://your-domain.com/api/v1/messages?target=MyChannel&search=Bitcoin&download_media=true" \
     -H "x-api-key: YOUR_API_KEY"

N8N Integration

Use the HTTP Request node in N8N with these settings:

  1. Method: GET
  2. URL: https://your-domain.com/api/v1/messages
  3. Query Params: target = your channel, limit = 10
  4. Headers: x-api-key = your secret key

Auto-Forwarder

Fetch latest message every minute. If new, forward to your group via Telegram Bot.
